# regularproperties
1. **Регулярные выражения** — шаблоны для поиска и обработки текста. Эффективны для поиска email, телефонов, дат, валидации данных.
2. **import re**
3. **re.search()** возвращает первое совпадение или None. re.findall() возвращает список всех совпадений.
4. **re.match()** — ищет совпадение только в начале строки.
5. **.** означает любой символ кроме новой строки.
6. **Группировка** — выделение части паттерна в скобки (). Позволяет извлечь только нужную часть совпадения. Например r'ID-(\d+)' извлечёт только цифры после ID-.
